# Known limitations

- The pilot corpus contains 100 abstracts from one biomedical domain.
- Rhetorical-role labels were generated by an LLM and validated in the unstructured subset.
- Both embedding models used in the main analysis are general-domain sentence-transformer models.
- Positional and lexical baselines recover part of the same review-clinical signal.
- Similarity values are descriptive, not normative.
- The framework is not an automated quality score and should not be used as one.
